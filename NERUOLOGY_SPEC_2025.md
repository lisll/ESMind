# 神经科数据调取规范 (2025年)

> 数据来源: ES 医疗数据仓库
> 说明: 本规范定义了神经科数据调取的入组条件、变量内容及提取路径，供 Agent 在构建 ES 查询时参考。

## 总体说明

ES 中存储的都是医疗行业数据。ES 的 mapping 结构基于**抽取模型**和**后结构化模型**建立：
- **抽取模型** — 用于普通表，如病案首页、门诊就诊记录等
- **后结构化模型** — 用于文书表，如病历文书和检查报告等

模型有多个版本（如 V2.0-20260507, V2.0-20260506, V2.0-20260422 等），每个版本包含多个表（检查报告、住院医嘱、门诊西医处方、住院诊断、住院手术、门诊诊断等）。模型中规定了每个字段的查询路径和含义。

## 一、入组条件
